# Portafolio Personal

Este es un portafolio personal desarrollado con Html5, Bootstrap y css

## Tecnologia Usadas

- Html5
- Css3
- Bootstrap
- Php
- Mysql
- Git

## Estructura del proyecto

- index.html: pagina principal
- css/main.css: estilo de la pagina web
- img/fondo.jpg: Imagen de fondo
- php: Carpeta para los archivos php

## Instalación

1. Clona el repositorio o descarga los archivos.
2. Coloca la carpeta en tu servidor local (por ejemplo, XAMPP).
3. Accede a `index.html` desde tu navegador.
